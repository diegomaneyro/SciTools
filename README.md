# 🔬 SciTools

> A collection of Python tools for Mathematics, Physics and Engineering.

SciTools is an open-source repository containing small Python applications designed to solve mathematical, physical, and engineering problems.

The objective is to build a practical library of calculators, numerical methods, simulations, and engineering utilities that can be used for learning, research, and professional work.

---

## 🚀 Features

- Pure Python implementations
- Well-documented source code
- Educational examples
- Numerical methods
- Engineering calculators
- Scientific computing
- Easy to extend
